# Заметка по проекту от 15.02.2026 

## Проектная тема: "Векторизация КТ данных"

### Основные вопросы на данном этапе:
- Какие данные считать целевыми для нашей задачи (КТ внутренних органов человека например или же КТ более простых объектов по типу грецкого ореха и тд)
- Основываясь на целевых данных необходимо решить какие методы стоит применять при разработке решения
- Определение черного ящика проекта (с чем мы работаем и что должно получаться на выходе рабочего решения)

### Контекст
В теме проекта напрямую упомянаются КТ данные и понимание их сути важно для последующего выполнения проекта. 
```
КТ (Компьютерная томография) - используется для изучения внутренней структуры объектов при помощи рентгеновского излучения. КТ показывает некую плотность материала внутри объекта по серии рентгеновских проекций, полученных под разными углами.
```
На выходе КТ получается 3D объем вокселей с числом, пропорциональным ослаблению рентгена. Конкретное предсавление же данного объема зависит от выбранного источника данных и целевого объекта. В общем смысле можно представить как стек последовательных двумерных срезов (поперечных сечений объекта). Именно эти данные и сейчас будем считать как входные для проекта.

На выходе должен получаться триангулированный меш целевого объекта, полученный из входных КТ данных. Меш представляет собой набор точек и ребер которые ограничивают объект, те его векторное представление в 3х мерном пространстве.

### Черный ящик
**Вход**:
Трёхмерный КТ объём объекта (стек поперечных срезов), ориентация, метаданные сканирования.

**Выход**:
Векторное 3D представление геометрии объекта в виде набора меш-моделей
